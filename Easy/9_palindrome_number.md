## Problem Summary
Given an integer `x`, determine whether it reads the same forward and backward.
If it does, return `true`; otherwise, return `false`.

Negative numbers and numbers ending with zero (except zero itself)
cannot be palindromes.

## Examples

**Example 1**  
Input: x = 121  
Output: true  

**Example 2**  
Input: x = -121  
Output: false  

**Example 3**  
Input: x = 10  
Output: false
